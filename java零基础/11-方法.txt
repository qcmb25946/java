方法
1.注意：
	- 程序开始执行的时候先执行main方法
	- 因为main方法是一个入口
	- 在java语言中所有的方法体都必须遵循自上而下的顺序逐行执行

2.方法怎么定义？语法机制是什么？
	[修饰符列表] (返回值类型) 方法名(形式参数列表){
		方法体;
	}
	注意：修饰符是可选项；方法体由java语句构成
	
	2.1 关于修饰符列表
		修饰符是可选项。目前统一写成：public static
	
	2.2 关于返回值类型
		- 返回值类型可以是任何数据类型；只要是java合法数据类型就行;数据类型包括基本数据类型和引用数据类型
		- 返回值一般指的一个方法执行结束之后的结果；结果通常是一个数据；所以被称为"值"；而且还叫做"返回值"
		- 当一个方法结束后不返回任何值的时候；返回值类型不能空白，必须写上viod关键字，void表示方法执行结束后不做返回
		- 如果返回值类型"不是void"，那么你在方法体执行结束的时候必须使用"return"值；这样的语句来完成"值"的返回，如果没有"return"值；呢么编译器报错
		- return值："返回值" 返回方法执行的结果
		- 只要有"return"关键字的语句执行，当前方法执行结束
		- 如果返回值类型是void，呢么在方法体当中不能有"return"值；这样的语句。但是可以有"return"语句。这个语句是用来终止当前方法的
		- 出了"void"以外，剩下的都必须有"return"值；这样的语句

	1.3 方法名
		- 方法名要见名知义。
		- 方法名在标识符命名规范中，要求首字母小写，后面每个首字母大写
		- 只要是合法的标志符就行。

	1.4 形式参数列表
		- 简称形参
		- 形式参数列表中的每一个参数都是"局部变量"，方法方法结束之后内存释放
		- 形参的个数是0-N个
		- 形参有多个使用逗号隔开。逗号是英文的
		- 形参的数据类型起决定性作用，形参对应的变量名是随意的

	1.4 方法体
		- 由java语句构成，java语句以";"结尾
		- 方法体当中编写的是业务逻辑代码,完成某个特定功能
        - 在方法体中处理业务逻辑代码的时候需要数据，数据来源形参

    1.5 方法定义之后怎么调用
        - 方法必须调用才能执行
        -调用语法：类名.方法名(实际参数列表);(在同一个类中类名.可以省略)
